## Οι 10 Βασικές Λέξεις για Κάθε Ραντεβού

Αυτές οι δέκα λέξεις είναι απαραίτητες για κάθε ρομαντική βραδιά. Μάθετέ τις μαζί και θα δείτε πόσο εύκολο γίνεται να εκφράσετε τα συναισθήματά σας!

| **Abbraccio** | Αγκαλιά | ahb-BRAH-choh | Για να ζητήσετε μια αγκαλιά |

## Λέξεις για το Ρομαντικό Δείπνο

Το ιταλικό φαγητό είναι συνώνυμο του έρωτα. Αυτές οι λέξεις θα σας βοηθήσουν να δημιουργήσετε την τέλεια ατμόσφαιρα:

<VocabCard
  word="Ristorante"
  translation="Εστιατόριο"
  pronunciation="ree-stoh-RAHN-teh"
  example="Prenotiamo un tavolo nel ristorante - Κλείνουμε τραπέζι στο εστιατόριο"
/>

| Ιταλικά | Ελληνικά | Προφορά |
|--------|----------|----------|
| **Cena** | Δείπνο | CHEH-nah |
| **Vino** | Κρασί | VEE-noh |
| **Pane** | Ψωμί | PAH-neh |
| **Acqua** | Νερό | AH-kwah |
| **Antipasto** | Ορεκτικό | ahn-tee-PAH-stoh |
| **Primo** | Πρώτο πιάτο | PREE-moh |
| **Secondo** | Κυρίως πιάτο | seh-KOHN-doh |
| **Dolce** | Επιδόρπιο | DOHL-cheh |
| **Conto** | Λογαριασμός | KOHN-toh |
| **Grazie** | Ευχαριστώ | GRAH-tsee-eh |

<CultureTip flag="🇮🇹" title="Ιταλική Τάξη Φαγητού">
Στην Ιταλία, το δείπνο έχει συγκεκριμένη σειρά: πρώτα antipasto, μετά primo (συνήθως ζυμαρικά), μετά secondo (κρέας ή ψάρι) και τέλος dolce. Μην βιαστείτε - οι Ιταλοί απολαμβάνουν αργά γεύματα!
</CultureTip>

## Λέξεις για Ρομαντικά Σχόλια

Μάθετε να εκφράζετε την αγάπη και τον θαυμασμό σας με αυτές τις γλυκιές λέξεις:

- **Bellissima** (πανέμορφη) - behl-LEE-see-mah
- **Meravigliosa** (θαυμάσια) - meh-rah-veel-YOH-zah
- **Incantevole** (μαγευτική) - een-cahn-TEH-voh-leh
- **Straordinaria** (εξαιρετική) - strah-ohr-dee-NAH-ree-ah
- **Speciale** (ιδιαίτερη) - speh-chah-LEH
- **Unica** (μοναδική) - OO-nee-kah

</CultureTip>

## Λέξεις για το Τέλος της Βραδιάς

Κλείστε το ραντεβού σας με στυλ:

- **Buonanotte** (καληνύχτα) - bwoh-nah-NOHT-teh
- **Sogni d'oro** (γλυκά όνειρα) - SOH-nyee DOH-roh
- **A domani** (μέχρι αύριο) - ah doh-MAH-nee
- **Sei stato meraviglioso** (ήσουν υπέροχος) - say STAH-toh meh-rah-veel-YOH-zoh
- **Grazie per questa serata** (ευχαριστώ για αυτό το βράδυ) - GRAH-tsee-eh pehr KWEH-stah seh-RAH-tah

4. **Προσθέστε χειρονομίες** - Τα ιταλικά είναι γλώσσα του σώματος, οπότε χρησιμοποιήστε τα χέρια σας!
