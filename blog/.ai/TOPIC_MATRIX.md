## Priority Legend

- **P0** = Core set, must have for all pairs (10 topics)
- **P1** = High value, generate second (30 topics)
- **P2** = Complete coverage, generate last (38 topics)

---

## Generation Order

1. **Phase 1:** P0 topics × 6 new native languages (60 × 17 = 1,020 articles)
2. **Phase 2:** P0 topics × gaps in existing 12 natives
3. **Phase 3:** P1 topics × all 18 natives
4. **Phase 4:** P2 topics × all 18 natives
