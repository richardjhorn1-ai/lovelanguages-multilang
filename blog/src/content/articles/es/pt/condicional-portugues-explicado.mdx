El condicional en portugues, tambien llamado "futuro do preterito", es esencial para expresarte con cortesia, hablar de deseos y plantear situaciones hipoteticas. Es una herramienta fundamental para comunicarte con elegancia con tu pareja.

## La Expresion del Dia

<PhraseOfDay
  word="Eu gostaria de te conhecer melhor"
  translation="Me gustaria conocerte mejor"
  pronunciation="eu gos-ta-RI-a de te co-nhe-SER me-LHOR"
  context="Una forma educada y romantica de expresar interes"
/>

## Formacion del Condicional

El condicional en portugues se forma de manera muy similar al espanol: infinitivo + terminaciones.

<ConjugationTable
  verb="amar (condicional)"
  meaning="amar"
  conjugations={[
    { person: "eu", word: "amaria", translation: "yo amaria" },
    { person: "tu", word: "amarias", translation: "tu amarias" },
    { person: "voce", word: "amaria", translation: "usted amaria" },
    { person: "ele/ela", word: "amaria", translation: "el/ella amaria" },
    { person: "nos", word: "amariamos", translation: "nosotros amariamos" },
    { person: "voces", word: "amariam", translation: "ustedes amarian" },
    { person: "eles/elas", word: "amariam", translation: "ellos/ellas amarian" }
  ]}
/>

<CultureTip
  title="Condicional vs Imperfeito"
  flag="ðŸ‡§ðŸ‡·"
>
  En el portugues brasileno hablado, es muy comun usar el imperfecto en lugar del condicional para expresar deseos. "Eu queria" (yo queria) se usa frecuentemente en lugar de "Eu quereria" (yo querria). Ambas formas son correctas.
</CultureTip>

## Usos del Condicional

### 1. Expresar Deseos

  word="Eu adoraria viajar com voce"

  pronunciation="eu a-do-ra-RI-a vi-a-JAR com vo-SE"

  title="Condicional en el Romance"
  flag="ðŸ‡µðŸ‡¹"
>
  El condicional es perfecto para expresar suenos y planes futuros con tu pareja de manera delicada. Decir "Gostaria de casar com voce" (Me gustaria casarme contigo) suena mas romantico y menos directo que "Quero casar com voce" (Quiero casarme contigo).
</CultureTip>
