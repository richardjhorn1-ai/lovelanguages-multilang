# Основи нідерландської граматики для початківців

Нідерландська мова — германська, родичка англійської та німецької. Для українців деякі концепції будуть новими, але є й спільне.

## Роди іменників

В нідерландській є два граматичні роди: **загальний** (de-слова) та **середній** (het-слова).

### De-слова (загальний рід)
Приблизно 75% іменників:
- de man (чоловік)
- de vrouw (жінка)
- de tafel (стіл)
- de zon (сонце)

### Het-слова (середній рід)
Приблизно 25% іменників:
- het kind (дитина)
- het huis (дім)
- het boek (книга)
- het water (вода)

**Порада:** Немає чіткого правила — вчіть артикль разом зі словом!

## Артиклі

### Означені артиклі

| Однина | Множина |
|--------|---------|
| de (загальний) | de |
| het (середній) | de |

У множині завжди **de**!

### Неозначені артиклі

| | Однина |
|--|--------|
| De-слова | een man (якийсь чоловік) |
| Het-слова | een huis (якийсь дім) |

Неозначений артикль завжди **een** (вимовляється «ен»).

## Дієслова: теперішній час

### Стандартна форма (werken — працювати)

| Особа | Дієслово | Переклад |
|-------|----------|----------|
| ik | werk | я працюю |
| jij/je | werkt | ти працюєш |
| hij/zij/het | werkt | він/вона працює |
| wij/we | werken | ми працюємо |
| jullie | werken | ви працюєте |
| zij/ze | werken | вони працюють |

### Правило

- **Основа** = інфінітив мінус -en
- **ik** = основа
- **jij/hij/zij** = основа + t
- **Множина** = інфінітив

## Важливі неправильні дієслова

### Zijn (бути)
- ik ben — я є
- jij bent — ти є
- hij/zij is — він/вона є
- wij zijn — ми є
- jullie zijn — ви є
- zij zijn — вони є

### Hebben (мати)
- ik heb — я маю
- jij hebt — ти маєш
- hij/zij heeft — він/вона має
- wij hebben — ми маємо
- jullie hebben — ви маєте
- zij hebben — вони мають

### Willen (хотіти)
- ik wil
- jij wilt
- hij/zij wil
- wij willen
- jullie willen
- zij willen

## Порядок слів

### Стверджувальне речення
**Підмет + Дієслово + Решта**
- Ik werk vandaag (Я працюю сьогодні)

### Питання
**Дієслово + Підмет + Решта**
- Werk jij vandaag? (Ти працюєш сьогодні?)

### Інверсія
Якщо речення починається НЕ з підмета, дієслово все одно друге:
- **Vandaag** werk **ik** (Сьогодні я працюю)

Це важливе правило! Дієслово завжди на другому місці.

## Заперечення

Додайте **niet** (для дієслів/прикметників) або **geen** (для іменників з een):

- Ik werk **niet** — Я не працюю
- Ik heb **geen** tijd — Я не маю часу
- Hij is **niet** blij — Він не щасливий

## Прикметники

### Перед іменником

Прикметники отримують закінчення **-e** (майже завжди):

| | de-слова | het-слова |
|--|----------|-----------|
| Означ. артикль | de mooie vrouw | het mooie huis |
| Неозн. артикль | een mooie vrouw | een mooi huis* |

*Виняток: het-слова з *een* — без закінчення!

### Після дієслова

Без закінчення:
- De vrouw is **mooi** (Жінка красива)

## Присвійні займенники

| Займенник | Нідерландська |
|-----------|---------------|
| мій | mijn |
| твій | jouw / je |
| його | zijn |
| її | haar |
| наш | ons / onze |
| ваш | jullie |
| їхній | hun |

## Питальні слова

| Нідерландська | Українська |
|---------------|------------|
| Wie? | Хто? |
| Wat? | Що? |
| Waar? | Де? |
| Wanneer? | Коли? |
| Waarom? | Чому? |
| Hoe? | Як? |
| Hoeveel? | Скільки? |

## Зменшувальні суфікси

В нідерландській дуже активно використовують суфікс **-je**:

- huis → huis**je** (будиночок)
- boek → boek**je** (книжечка)
- kind → kind**je** (дитинко)

**Для українців:** Схоже на наші зменшувальні суфікси!

## Корисні конструкції для пар

- **Ik hou van + іменник** — Я люблю...
- **Ik vind je + прикметник** — Я вважаю тебе... / Ти для мене...
- **Ik wil + інфінітив** — Я хочу...
- **Mag ik + інфінітив?** — Чи можу я...?
- **Kun je + інфінітив?** — Ти можеш...?

## Подібність до англійської

Багато слів схожі:

| Нідерландська | Англійська | Українська |
|---------------|------------|------------|
| water | water | вода |
| huis | house | дім |
| boek | book | книга |
| man | man | чоловік |
| goed | good | добре |

Це допоможе у вивченні!
