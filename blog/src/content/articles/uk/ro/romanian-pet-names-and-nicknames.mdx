# Румунські прізвиська та ласкаві імена для коханих

Румуни — майстри створювати теплі та оригінальні прізвиська. Їхня мова багата на зменшувальні форми, що робить кожне звертання особливим.

## Класичні прізвиська

### Frumos/Frumoasă (Фрумос/Фрумоасе)
Означає «красивий/красива». Дуже поширене.

- **Frumoaso!** — Красуне!
- **Frumusețea mea** — Моя красо
- **Cel/Cea mai frumos/ă** — Найкрасивіший/найкрасивіша

### Drag/Dragă (Драг/Драге)
Означає «дорогий/дорога».

- **Draga mea** — Моя дорога
- **Drăguțule/Drăguțo** — Любий/Люба
- **Drăgălașule** — Любенький

## Тваринні прізвиська

Румуни люблять милі порівняння з тваринками:

| Румунська | Вимова | Значення |
|-----------|--------|----------|
| Puiuț | Пуюц | Пташенятко |
| Pisicuță | Пісікуце | Кішечка |
| Iepuraș | Єпураш | Зайчик |
| Ursuleț | Урсулец | Ведмедик |
| Căprioara | Кепріоара | Козуля |
| Porumbița | Порумбіца | Голубка |

### Puiuț — найпопулярніше

**Puiuț** (пташеня, курчатко) — це, мабуть, найніжніше румунське звертання:

- **Puiuțul meu** — Моє пташенятко
- **Puiuțo** — Пташенятко (звертання)
- **Puișor** — Пташечка

## Космічні та природні прізвиська

- **Steluța mea** (Стелуца мя) — Моя зірочка
- **Soarele meu** (Соареле меу) — Моє сонце
- **Luna mea** (Луна мя) — Мій місяць
- **Floricica mea** (Флорічіка мя) — Моя квіточка
- **Îngerașul meu** (Инджерашул меу) — Мій янголятко

## Солодкі прізвиська

- **Bomboană** (Бомбоане) — Цукерка
- **Dulceață** (Дулчаце) — Солодкість
- **Prăjitură** (Пражітуре) — Тістечко (жартівливо)
- **Ciocolată** (Чоколате) — Шоколадка

## Прізвиська від імен

Румунські імена легко перетворюються на пестливі форми:

**Чоловічі:**
- Alexandru → Sandu, Alex, Alecu
- Andrei → Andruț, Andruțu
- Ion → Ionuț, Ionel, Nelu
- Mihai → Mișu, Mitica

**Жіночі:**
- Maria → Măriuca, Mia, Mara
- Elena → Lenu, Lenuța
- Ana → Anuța, Ani
- Andreea → Anda, Dea

## Жартівливі прізвиська

Румуни люблять жартівливі звертання з любов'ю:

- **Nebunule/Nebuno** — Божевільний/божевільна
- **Prostulețul meu** — Мій дурничок
- **Gâgâlice** — Дурненька
- **Monstru mic** — Маленький монстр
- **Trăsnitul meu** — Мій чудик

## Сучасні прізвиська

Молодь також використовує:

- **Baby** / **Bebe** — малюк
- **Iubi** — від iubire (любов)
- **Prințesă/Prinț** — Принцеса/Принц
- **Boss** — жартівливо для хлопця

## Регіональні особливості

**Трансільванія:**
- Більш стриманий стиль
- Drăguță, Scumpă

**Молдова (румунська):**
- Більш емоційний
- Mândruță, Puiule

**Олтенія та Мунтенія:**
- Puiuț, Suflete, Iubire

## Суфікси ніжності

Румунська, як і українська, багата на зменшувальні суфікси:

| Суфікс | Приклад |
|--------|---------|
| -uț/-uță | drag → drăguț |
| -ișor/-ișoară | pui → puișor |
| -el/-ea | frate → frățel |
| -ică/-ico | floare → floricică |

## Як обрати прізвисько

1. **Починайте м'яко** — Iubire, Dragă, Scumpă
2. **Додавайте зменшувальні суфікси** — це завжди працює
3. **Слухайте партнера** — що йому/їй подобається?
4. **Створюйте спільні** — внутрішні жарти стають найкращими прізвиськами

## Порада

Румуни дуже цінують, коли іноземці намагаються говорити румунською. Навіть просте *puiuț* чи *iubire* зробить вашого партнера щасливим!

Пам'ятайте: найкращі прізвиська народжуються з ваших спільних моментів та історій.
