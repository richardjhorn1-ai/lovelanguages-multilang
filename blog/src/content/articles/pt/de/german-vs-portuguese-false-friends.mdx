Embora alemao e portugues nao sejam linguas irmas, existem palavras que parecem familiares mas tem significados completamente diferentes. Conhecer esses "falsos amigos" vai evitar mal-entendidos constrangedores!

## Falsos Amigos Mais Perigosos

<VocabCard
  word="Gift"
  translation="Veneno (NAO presente!)"
  pronunciation="[GIFT]"
  example="Achtung! Dieses Produkt enthalt Gift. (Cuidado! Este produto contem veneno.)"
/>

**IMPORTANTE:** "Gift" em alemao significa VENENO! Para "presente", use **Geschenk**.

<VocabCard
  word="Rat"
  translation="Conselho (NAO rato!)"
  pronunciation="[RAT]"
  example="Ich brauche deinen Rat. (Preciso do seu conselho.)"
/>

Para "rato", use **Ratte** ou **Maus** (camundongo).

<PhraseOfDay
  word="Ich habe einen guten Rat fur dich"
  translation="Tenho um bom conselho para voce"
  pronunciation="[ich HA-be AI-nen GU-ten RAT fur dich]"
  context="Nada a ver com roedores!"
/>

<CultureTip title="Origem das Palavras" flag="ðŸ‡©ðŸ‡ª">
Embora alemao e portugues tenham raizes indo-europeias comuns, o alemao e germanico e o portugues e romanico. Palavras parecidas geralmente sao coincidencias ou emprestimos de outras linguas.
</CultureTip>

## Falsos Amigos com Ingles

Muitos brasileiros aprendem alemao depois do ingles, o que cria novas armadilhas:

<VocabCard
  word="bekommen"
  translation="Receber (NAO 'become'/tornar-se!)"
  pronunciation="[be-KO-men]"
  example="Ich bekomme einen Brief. (Eu recebo uma carta.)"
/>

Para "tornar-se", use **werden**.

<VocabCard
  word="brav"
  translation="Bem-comportado (NAO corajoso/bravo!)"
  pronunciation="[BRAF]"
  example="Das Kind ist sehr brav. (A crianca e muito bem-comportada.)"
/>

Para "corajoso", use **mutig** ou **tapfer**.

<VocabCard
  word="Gymnasium"
  translation="Escola secundaria (NAO academia!)"
  pronunciation="[gum-NA-zi-um]"
  example="Mein Sohn geht aufs Gymnasium. (Meu filho frequenta o ensino medio.)"
/>

Para "academia" (de ginastica), use **Fitnessstudio**.

## Tabela de Falsos Amigos Essenciais

| Alemao | Parece | Significa Realmente |
|--------|--------|---------------------|
| **Gift** | presente | veneno |
| **Rat** | rato | conselho |
| **brav** | bravo | bem-comportado |
| **Chef** | chef | chefe/patrao |
| **fast** | rapido | quase |
| **also** | tambem | portanto/entao |
| **aktuell** | atual | atual (correto!) |
| **Fabrik** | fabrica | fabrica (correto!) |
| **Karte** | carta | mapa/cardapio |
| **Mappe** | mapa | pasta (documentos) |

<CultureTip title="Chef vs. Koch" flag="ðŸ‡©ðŸ‡ª">
"Chef" em alemao significa chefe/patrao. Para "chef de cozinha", use **Koch** (cozinheiro) ou **Chefkoch** (chef de cozinha). No restaurante, pergunte pelo Koch!
</CultureTip>

## Mais Falsos Amigos Comuns

<VocabCard
  word="fast"
  translation="Quase (NAO rapido!)"
  pronunciation="[FAST]"
  example="Ich bin fast fertig. (Estou quase pronto.)"
/>

Para "rapido", use **schnell**.

<VocabCard
  word="also"
  translation="Portanto/Entao (NAO 'tambem'!)"
  pronunciation="[AL-zo]"
  example="Also, was machen wir jetzt? (Entao, o que fazemos agora?)"
/>

Para "tambem", use **auch**.

<VocabCard
  word="Karte"
  translation="Mapa/Cardapio/Ingresso (depende do contexto)"
  pronunciation="[KAR-te]"
  example="Kann ich die Karte sehen? (Posso ver o cardapio?)"
/>

**Karte** pode significar:
- Landkarte = mapa
- Speisekarte = cardapio
- Eintrittskarte = ingresso
- Kreditkarte = cartao de credito

Para "carta" (correspondencia), use **Brief**.

## Exercicio para Casais

### Jogo dos Falsos Amigos
1. Um parceiro diz uma palavra em alemao
2. O outro deve adivinhar se e falso amigo ou cognato verdadeiro
3. Se for falso, explique a diferenca
4. Riam juntos das possiveis confusoes!

### Criando Historias
Inventem historias engracadas usando falsos amigos:
- "Ele deu um Gift (veneno) de aniversario..."
- "O Rat (conselho) estava no restaurante..."
