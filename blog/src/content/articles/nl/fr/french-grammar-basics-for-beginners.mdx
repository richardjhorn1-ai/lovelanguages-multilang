Frans heeft de reputatie een lastige taal te zijn, maar als Nederlander heb je een voordeel: je kent waarschijnlijk al wat Frans van school! Deze gids legt de basis helder uit, met aandacht voor de verschillen met Nederlands.

## Zelfstandige Naamwoorden: Mannelijk en Vrouwelijk

<PhraseOfDay
  word="Le / La"
  translation="De (mannelijk) / De (vrouwelijk)"
  pronunciation="luh / la"
  context="Frans heeft twee geslachten, net als Nederlands 'de' en 'het'."
/>

In het Frans is elk zelfstandig naamwoord mannelijk (masculin) of vrouwelijk (feminin):

### Vuistregels

| Regel | Mannelijk (le) | Vrouwelijk (la) |
|-------|----------------|-----------------|
| Eindigt op medeklinker | le restaurant | - |
| Eindigt op -e | - | la table |
| Eindigt op -tion/-sion | - | la nation |
| Eindigt op -age | le voyage | - |
| Eindigt op -ment | le moment | - |

<VocabCard
  word="L'amour"
  translation="De liefde (mannelijk!)"
  pronunciation="la-moor"
  example="L'amour est beau. (De liefde is mooi.)"
/>

<CultureTip title="Let op!" flag="âš ï¸">
Er zijn veel uitzonderingen! "Le probleme" is mannelijk ondanks de -e. Leer woorden altijd met hun lidwoord (le/la).
</CultureTip>

## Meervoud

Het meervoud is meestal eenvoudig - voeg een -s toe:

| Enkelvoud | Meervoud | Regel |
|-----------|----------|-------|
| le livre | les livres | +s |
| la table | les tables | +s |
| le gateau | les gateaux | -eau â†’ -eaux |
| le journal | les journaux | -al â†’ -aux |

## Tu vs Vous

<VocabCard
  word="Tu / Vous"
  translation="Jij / U (of jullie)"
  pronunciation="tue / voo"
  example="Tu = informeel, Vous = formeel of meervoud"
/>

| Gebruik TU voor: | Gebruik VOUS voor: |
|------------------|-------------------|
| Vrienden | Onbekenden |
| Familie | Ouderen |
| Kinderen | In professionele situaties |
| Huisdieren | Eerste ontmoetingen |

<CultureTip title="Vouvoyer" flag="ðŸŽ©">
Begin ALTIJD met "vous" bij volwassenen die je niet kent. Als zij "on peut se tutoyer?" (kunnen we tutoyeren?) zeggen, schakel je over naar "tu."
</CultureTip>

## Ontkenning

Ontkenning in het Frans gaat met **ne...pas** om het werkwoord:

<VocabCard
  word="Je ne comprends pas"
  translation="Ik begrijp het niet"
  pronunciation="zjuh nuh koh-proh pa"
  example="NE voor het werkwoord, PAS erna!"
/>

| Bevestigend | Ontkennend |
|-------------|------------|
| Je parle | Je ne parle pas |
| J'aime | Je n'aime pas |
| Il mange | Il ne mange pas |

**In spreektaal:** De "ne" wordt vaak weggelaten: "Je comprends pas"

## Vragen Stellen

### Drie Manieren

1. **Intonatie** (makkelijkst): Tu viens? (â†—)
2. **Est-ce que**: Est-ce que tu viens?
3. **Inversie** (formeel): Viens-tu?

- **G**oodness: bon/bonne, mauvais

## Lidwoorden: Samentrekkingen

| Voorzetsel + le | Samentrekking |
|-----------------|---------------|
| a + le | au |
| a + les | aux |
| de + le | du |
| de + les | des |

## Tips voor Succes

1. **Leer woorden met lidwoord** - "la table" niet alleen "table"
2. **Luister naar uitspraak** - Eindletters zijn vaak stom
3. **Begin met tu/vous goed** - Formeel met onbekenden
4. **Oefen vervoegingen** - Ze komen terug, elke keer!
5. **Lees hardop** - Frans klinkt mooier dan het er uitziet
