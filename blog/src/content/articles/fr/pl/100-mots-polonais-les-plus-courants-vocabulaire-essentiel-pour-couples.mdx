## Les pronoms personnels : la base fondamentale

Comprendre les **zaimki osobowe** (pronoms personnels) est crucial pour toute conversation :

- **Ja** (je) - se prononce "ya"
- **Ty** (tu, informel) - se prononce "ti"
- **On** (il) - se prononce "one"
- **Ona** (elle) - se prononce "O-na"
- **Ono** (il/elle neutre) - se prononce "O-no"
- **My** (nous) - se prononce "mi"
- **Wy** (vous, pluriel/formel) - se prononce "vi"
- **Oni/One** (ils/elles) - se prononce "O-ni/O-né"

<CultureTip title="Formel vs. Informel">
Utilisez **ty** (tu) avec vos amis, votre famille et votre partenaire. Utilisez **Pan** (Monsieur) ou **Pani** (Madame) dans les situations formelles. En cas de doute, commencez par le formel !
</CultureTip>

## Les expressions de politesse

Expressions polies que chaque couple devrait maîtriser :

- **Proszę** (s'il vous plaît/je vous en prie) - "PRO-ché"
- **Dziękuję** (merci) - "djién-KOU-yé"
- **Przepraszam** (excusez-moi/pardon) - "pché-PRA-cham"
- **Miło mi Cię poznać** (enchanté de vous rencontrer) - "MI-ouo mi tchié POZ-natch"
- **Na zdrowie** (santé/à vos souhaits) - "na ZDRO-vié"
- **Dobranoc** (bonne nuit) - "do-BRA-nots"

<CultureTip title="L'accord des adjectifs">

</CultureTip>

## Votre stratégie d'apprentissage

En tant que couple apprenant ensemble :

1. **Pratiquez quotidiennement** - Utilisez 5-10 nouveaux mots en conversation chaque jour
2. **Testez-vous mutuellement** - Rendez cela amusant avec des jeux et des défis
3. **Utilisez le contexte** - Ne mémorisez pas seulement ; utilisez les mots dans des situations réelles
4. **Soyez patients** - La grammaire polonaise est complexe, mais ces mots de base vous serviront bien
