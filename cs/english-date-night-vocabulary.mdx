# Anglická slovíčka pro rande

## Tipy pro romantický večer

1. **Buďte přítomní** - Dejte telefon stranou a soustřeďte se na partnera
2. **Mluvte o pozitivních věcech** - Sdílejte sny, plány, vzpomínky
3. **Poslouchejte** - Ptejte se a skutečně poslouchejte odpovědi
4. **Dělejte komplimenty** - Upřímné a konkrétní
5. **Užijte si okamžik** - Neuspěchávejte nic
