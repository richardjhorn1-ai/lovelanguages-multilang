# Design Skill

This skill provides the Love Languages design system rules. Load it when doing UI work — creating components, styling, responsive design, or reviewing visual consistency.

## Rules

See `rules/` directory for topical guidance:

| File | Covers |
|------|--------|
| `principles.md` | Design philosophy, avoiding AI slop, when adding new UI |
| `colors.md` | Color palette, tier colors, target word highlight, background |
| `typography.md` | Outfit + Quicksand fonts, Tailwind classes |
| `components.md` | Cards, buttons, badges, progress indicators, icon system |
| `mobile.md` | Breakpoints, size scales, nav, panels, scrollbar, grid layouts |
| `motion.md` | Animation guidelines, CSS-only preference, interaction patterns |
